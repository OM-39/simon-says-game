# Simon Says Game 🎮

A fun little memory game I built while practicing JavaScript. The idea is simple — watch the pattern, repeat it back, and try not to mess up as it gets longer!

---

## What is this?

It's the classic Simon Says game. Four colored buttons light up in a sequence, and you have to click them in the same order. Each round adds one more step to the sequence. The further you get, the harder it is to keep track.

I built this to get more comfortable with DOM manipulation and event handling in vanilla JavaScript — no frameworks, no libraries, just plain HTML, CSS, and JS.

---

## How to play

1. Open `index.html` in your browser
2. Press any key to start
3. Watch which button flashes
4. Click the buttons in the same order
5. If you get it right, the next round adds one more flash
6. If you get it wrong... game over. Your score gets saved and you can try to beat it

## Features
